# Проект 4: Место

# Описание проекта

Проект Mesto представляет собой интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.

# Технологии

* HTML5, CSS3
* Grid Layout, Flex
* БЭМ, файловая структура Nested БЭМ
* JavaScript 6
